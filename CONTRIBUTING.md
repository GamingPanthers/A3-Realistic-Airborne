# Contributing to Realistic Airborne (RA)

First off, thank you for considering contributing to Realistic Airborne! It's people like you that make the Arma modding community great.

### üõ† How Can I Contribute?

1. **Reporting Bugs**: 
   - Check the GitHub Issues page or Steam comments to see if it's already been reported. 
   - Provide your RPT logs and steps to reproduce the error.

2. **Suggesting Enhancements**:
   - Have an idea for the Roadmap? (e.g., Jumpmaster logic or Ramp detection). 
   - Open an issue or contact GamingPanthers via Discord. 

3. **Code Contributions**:
   - Fork the repository.
   - Create a new branch for your feature or fix.
   - Use HEMTT to verify your changes with `hemtt check`. 
   - Ensure your SQF follows the project's optimization standards (use `objectParent` and `configOf`).

### üìú Style Guide
- Use the `RA_fnc_` tag for all functions.
- Keep logic stateless where possible to maintain performance. 
- Document your functions using standard headers.
